# Assets

This folder holds visual and supporting materials referenced by the chapters: diagrams, charts, calligraphy, illustration references, and infographics.

## The Halal-Imagery Rule (strict)

All imagery in this encyclopedia must be **halal**:

- ❌ **NO depiction** of Allah, the Prophets (peace be upon them), the angels, or the Sahaba.
- ✅ **Permitted:** light, the cosmos, Arabic calligraphy, landscapes, silhouettes, objects, architecture, maps, diagrams, timelines, and abstract/symbolic art.
