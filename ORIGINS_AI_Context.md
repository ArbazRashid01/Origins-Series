## THE DOCUMENTARY FORMULA (one per season)

**Runtime Rule:** *"As short as possible, as long as necessary."*

2. **Runtime Estimate**
3. **Documentary Overview** (1 paragraph)

5. **Introduction** — briefly intro Origins + where this season fits in the larger timeline
6. **Full Script in CHAPTERS** — each reel becomes one expanded chapter with:
   - Smooth transition from previous chapter
   - Added historical context and depth
   - Qur'an references (Qur'an FIRST)
   - Authentic Hadith references (second, with grading/collection)
   - Visual suggestions
   - Music/sound design suggestions
   - Maps, timelines, graphics suggestions
   - **"Why does this matter today?"** reflection segment at the end of each chapter
7. **Finale** — summarize season + reinforce the main lesson + create anticipation for next season
8. **Post-Credit Teaser (30–60s)** — teaser for the next season

## THE AUTHENTICITY RULES (THE SOUL OF THE SERIES)

These are non-negotiable. Every script must follow them.

### Source Hierarchy
1. **Qur'an FIRST** — always
2. **Authentic Hadith SECOND** — Sahih/Hasan, with collection + grading stated
3. Trusted classical scholars for interpretation only

### The Hard Lines
- ❌ **NEVER invent dialogue** — if the Qur'an or hadith didn't record the exact words, narrate *about* it, never script fake conversation
- ❌ **NEVER invent emotions or events** not supported by sources
- ⚠️ **Clearly label** on screen and in narration: weak narrations, Isra'iliyyat, and scholarly differences
- ✅ **If a detail is unknown — say it is unknown.** Present it as honest uncertainty.
- ✅ **Qur'anic recorded speech** is not "invented dialogue" — it can be quoted directly
- ⚠️ **Isra'iliyyat** (stories from earlier traditions) — 3 types:
  - Confirmed by Islam → usable
  - Contradicted by Islam → reject completely
  - Unconfirmed → mention ONLY as "narrated, but not authentically established" — never as fact

### Authenticity Grading Cheatsheet
| Grade | Meaning | Can use? |
|-------|---------|----------|
| Sahih | Authentic | ✅ Yes |
| Hasan | Good | ✅ Yes |
| Da'if | Weak | ❌ No (not for doctrine/facts) |
| Mawdu' | Fabricated | ❌ Never |
| Isra'iliyyat | From earlier traditions | ⚠️ Only if confirmed by Islam |

## KEY ACCURACY FLAGS (COMMON ERRORS TO AVOID)

These are the most common mistakes — always flag them:

| Topic | Common Wrong Version | Correct Islamic Position |
|-------|---------------------|--------------------------|
| Iblis | "Fallen angel" | He was a JINN, not an angel (Qur'an 18:50) |
| The type of Adam's tree | "Wheat" or "fig" | Unknown — not in Qur'an or sahih hadith (Isra'iliyyat) |
| Adam's blame | "Hawwa convinced Adam" | Qur'an addresses both; often singles out Adam (20:121) — no Hawwa-blame framework |
| Ayyub's illness | "Worms ate his body" | Not in the Qur'an — Isra'iliyyat |
| Isa (Jesus) | "Crucified" | Qur'an 4:157–158: he was NOT killed or crucified — Allah raised him |
| Isa's nature | "Son of God" or "divine" | He is Allah's servant and messenger (4:171; 19:30) |
| "Azrael" (Angel of Death) | Common name used freely | NOT firmly established in authentic sources — say "Angel of Death" (32:11) |
| The cave story (Hijrah) | "Spider wove a web + pigeon laid eggs" | This narration is weak — do not include it |
| Sulayman's queen | Named "Bilqis" | Unnamed in the Qur'an — "Bilqis" is traditional, must be noted |
| Nuh's son | Named in the story | Unnamed in the Qur'an — do not give a name |
